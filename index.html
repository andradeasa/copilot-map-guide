How to Update CoPilot Maps

Driver Quick-Reference Guide  |  DC 6816

⏱ ~15 min
1
Connect to Walmart Wi-Fi
📶
  • From the tablet home screen, open ☰ Menu
  • Tap Connection Manager
  • Select Wi-Fi Assist
  • Connect to Walmartwifi
🚫 Do NOT select any other Wi-Fi network
2
Open CoPilot
🗺️
  • Return to the home screen
  • Tap the CoPilot app icon
ℹ️ Make sure Wi-Fi is still connected before continuing
3
Start the Map Download
⬇️
  • Tap ☰ Menu in CoPilot
  • Select MyCoPilot → My Maps
  • Under North America, tap the white arrow
✅ Blue progress bar = download is active (10–15 min)
4
Troubleshoot: No Download?
⚠️
  • Verify Walmartwifi is connected
  • Close all apps (bottom-right button → Close All)
  • Start over from Step 1
⚠️ Only try this if no progress bar appears
5
Close & Reopen CoPilot
🔄
  • When download finishes, CoPilot returns to home
  • ☰ Menu → ⏻ Power to close CoPilot
  • Reopen the CoPilot app
6
Confirm the Update
✔️
  • ☰ Menu → Settings → About CoPilot
  • Verify Version / Year / Quarter / Build
✅ If it matches ops/trainer info — update is complete!
7
Disconnect from Wi-Fi
📴
  • Home screen ☰ Menu → Connection Manager
  • Tap Disconnect from Walmartwifi
8
Reconnect to the Tractor
🚛
  • Select Vehicle
  • Enter Tractor ID (NO dashes)
  • Tap CONNECT
🎉 Done — CoPilot is updated and ready to use!
